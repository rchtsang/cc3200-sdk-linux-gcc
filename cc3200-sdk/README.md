# Building for Linux

This is the CC3200 SDK v1.5.0 that has been modified to build on linux based on the work done by Siarhei Volkau. 

Not everything is working.

Only the following examples build successfully:
- aes
- blinky
- crc
- des
- freertos_demo
- wlan_station
- i2c_demo
- pwm
- sdhost
- sdhost_fatfs
- shamd5
- sleep
- spi_demo
- timer
- timer_cc
- uart_demo
- uart_dma
- udma
- watchdog
